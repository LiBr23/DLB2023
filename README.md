## 画像の準備
画像は容量の関係でGitHub上にアップロードしていないので、お手数ですが、下記からDLしていただきますようお願いします。(4GBあります)
> https://www.kaggle.com/datasets/crawford/cat-dataset

> /data/images/CAT_00/xxx.jpg

という形で格納ください。
